# NumPy

Numpyは数値計算をするためのパッケージ。
これを使用することで、ベクトルや行列計算ができるようになる。

NumPy stands for numerical python.
NumPyで整形したデータのパターンを機械学習アルゴリズムが分析する。

データを以下のようにして扱う。

```
# NumPy's main datatype is ndarray
a1 = np.array([1, 2, 3])
a1

array([1, 2, 3])
```
